# 🐕 Robot Dog Project & Personal Portfolio

Welcome to my personal portfolio repository! This project showcases my personal webpage built with HTML, featuring my main highlight: the **Robot Dog Project**.

---

## 📝 Project Description

The **Robot Dog Project** is an exciting exploration into the world of robotics and hardware integration. 
The goal of this project was to design and program a smart robotic dog capable of executing basic movements and responding to control systems. It combines hardware assembly, embedded systems programming, and control logic to bring the robot to life.

---

## 🛠️ Tech Stack & Skills
* **Hardware & Robotics:** Control systems, sensor integration, and hardware troubleshooting.
* **Web Development:** HTML5 & CSS3 (used to build this portfolio).
* **Problem Solving:** Logical design and system debugging.
